# 牛圈

最小量測範圍：0.01 lux

* VCC接3.3V,GND接地,SDA接A4,SCL接A5

# --- TSL2561 ---
* GY-2561 -> Arduino
* VCC -> 3.3V
* GND -> GND
* SDA -> A4
* SCL -> A5

# --- buzzer ---
* buzzer+ -> D8
* buzzer- -> GND
